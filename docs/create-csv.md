今、kenectというアプリを開発しています
これは、県や市区町村などを題材にして行うゲームです

流れ

1. ランダムにどこか1地域が選択されます
   日本モードなら都道府県が一つランダムで選ばれます
   東京モードなら東京の市区町村からランダムで一つ選ばれます
2. ユーザが選ばれた現在地に接する地域を一つ宣言します
3. 宣言通り、隣接していれば次のターンに進み、隣接しなければそのユーザの負けです

このゲームの情報収集をお願いしたいです
